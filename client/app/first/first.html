Create an object with the following properties:

fullName, email, skype, and github.

Each of these properties should have strings as their values, which correspond to your information. If you don't have a GitHub account, you may set its value as null.

function getContactInfo() { // Modify this object to include your name and email var info = { fullName: "Yusef Ouda", email: "yiouda92@gmail.com", skype: "yusef.ouda", github: "YusefOuda" }; return info; }
